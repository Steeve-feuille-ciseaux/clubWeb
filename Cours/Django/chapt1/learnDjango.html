Comment installer Django (depuis zéro)

1. Installer Python

Télécharge et installe Python 3 depuis python.org.

Vérifie l'installation avec :

python --version

2. Créer un environnement virtuel (optionnel)

Dans le dossier de ton projet, crée un environnement virtuel :

python -m venv env

Active-le :

  • Windows : .\env\Scripts\activate
  • macOS/Linux : source env/bin/activate

3. Installer Django

pip install django

4. Vérifier l'installation

django-admin --version

5. Créer un projet Django

django-admin startproject monprojet

6. Lancer le serveur

cd monprojet
python manage.py runserver

7. Ouvrir dans le navigateur

Visite http://127.0.0.1:8000 pour voir la page de bienvenue Django.

Résumé rapide

mkdir monprojet-django
cd monprojet-django
python -m venv env
# Windows
.\env\Scripts\activate
# macOS/Linux
source env/bin/activate
pip install django
django-admin startproject monprojet
cd monprojet
python manage.py runserver
