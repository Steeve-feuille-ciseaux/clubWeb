Django : Première page et base de données

1. Créer une première page

Étapes :

  1. Créer une application : 
    python manage.py startapp pages
  2. Déclarer l’application dans settings.py : 
    'pages',
  3. Créer une vue dans pages/views.py : 
    from django.http import HttpResponse

def home(request):
    return HttpResponse("Bonjour, Django !")
  4. Créer pages/urls.py : 
    from django.urls import path
from . import views

urlpatterns = [
    path('', views.home, name='home'),
]
  5. Ajouter les URLs de l'app dans monprojet/urls.py : 
    from django.urls import path, include

urlpatterns = [
    path('admin/', admin.site.urls),
    path('', include('pages.urls')),
]

Visiter : http://127.0.0.1:8000

2. Gérer une base de données

Créer un modèle dans pages/models.py :

from django.db import models

class Article(models.Model):
    titre = models.CharField(max_length=100)
    contenu = models.TextField()
    date_pub = models.DateTimeField(auto_now_add=True)

    def __str__(self):
        return self.titre

Créer la base de données :

python manage.py makemigrations
python manage.py migrate

Enregistrer le modèle dans l’admin :

from django.contrib import admin
from .models import Article

admin.site.register(Article)

Créer un superutilisateur :

python manage.py createsuperuser

Accès à l’administration : http://127.0.0.1:8000/admin
